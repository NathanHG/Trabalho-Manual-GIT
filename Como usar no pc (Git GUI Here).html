Como baixar o Github

Primeiramente acesse a página de download do github.

Para que vc possa fazer o download do github acesse esse link: Download Github

Baixando Github

Após acessar a página, escolha seu SO.

Passo-a-passo

Após ter escolhido seu SO, o github baixara automaticamente.

Passo-a-passo

Após ter clicado em cima do download, clique em abrir.

Passo-a-passo

Agora basta apertar em Executar, dar next e instalar no local desejado.
