Como usar o Git GUI Here

Apos ter criado tudo, vamos aprender a usar o Git GUI Here.

Para que vc possa criar sua conta no Github acesse este link: Github

Para que vc possa baixar o Git acesse este link: Github

Primeiramente vamos criar um repositório no Github.

Acesse Github e clique em start a project.

Configurando Repositório

Coloque o nome do repositório, selecione a opção public, se sua conta for free, não esqueça de selecionar o campo initialize this repository with a README, clique em create repository

Acessando repositório no PC.

Após ter criado um repositório, clique em clone or download e copie o link apresentado.

Acessando no Git

No menu iniciar, procure por Git GUI, e acessando clique em Clone Existing Repository.

Criando pasta

No campo Source Location cole o link copiado no Github.

No campo Target Directory selecione o local na qual você quer que sua pasta seja salva e digite o nome da pasta (Não coloque em uma pasta já criada, o propeio Git criará uma para você).

Clique em clone.

Configurando Git GUI

Continuando... complete com seu usuário no Github e seu email no Github os campos user name, email address e clique em save.

Mandando algum arquivo.

Agora vamos mandar um arquivo para nossa pasta no Github, quando vc tiver um arquivo na pasta clique em Rescan e em seuida em Stage Changed.

Comitando

Agora digite algo na caixa de texto abaixo e clique em Commit.

Push

Por fim clique em Push e em Push denovo.

Complete as caixas de texto do login e pronto(Caso ele pessa o login mais vezes fazer o logim.).
